# Trilha API

API desenvolvida em Java com Spring Boot para gerenciamento de alunos, professores e agendamentos.

O objetivo do projeto é criar uma API de agendamentos que conecta **Professor** e **Aluno**, permitindo cadastrar, listar e futuramente agendar horários com regras para evitar conflitos.

## Status do projeto

Em desenvolvimento.

Funcionalidades em andamento:

- Cadastro de aluno
- Listagem de alunos
- Validação de e-mail duplicado
- Conexão com banco MySQL via JDBC

Funcionalidades futuras:

- CRUD de professor
- CRUD de agendamento
- Validação de conflitos de horário
- Interface front-end em React
- Deploy da aplicação

## Tecnologias utilizadas

- Java
- Spring Boot
- JDBC / JdbcTemplate
- MySQL 8.4
- Docker
- Git e GitHub
- DBeaver
